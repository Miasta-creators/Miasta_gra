# Miasta_gra
Continuation of @ATcat-pl Miasta_gra. A tile game about building cities.

Requirements:
- Java 17
- Windows
- 100 mb ram
- 130+ mb free space for each randomly generated map
- versions older than 1.6 may not work without internet connection
